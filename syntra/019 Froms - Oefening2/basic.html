Methods

When an object has a property with a function as the value, it is referred to as a method of that object.

We've been using methods all along! For example, when we use console.log(), log is a method of the console object. When we use Math.random(), random() is a method of the Math object. While it is technically correct to say random and log are functions, when a function is associated only with a specific object using the term method is more specific since it implies association with an object, and not a global function.
